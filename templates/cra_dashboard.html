What Q-CRA Dashboard Does

Automated compliance assessment — from scan to signed PDF in seconds

📋

12 CRA Articles Mapped

Art.10, Art.11, Art.13, Art.14, Annex I — every essential cybersecurity requirement covered with specific scanner checks.

🔏

ECDSA P-384 Signed

Every report is digitally signed with ECDSA P-384 + SHA-384. Tamper detection built in. Verifiable with CLI.

⚛️

PQC Readiness Check

Detects missing post-quantum cryptography (ML-KEM, ML-DSA) and flags harvest-now-decrypt-later risk.

📊

Scoring Methodology

Transparent scoring: PASS=100%, WARNING=50%, FAIL=0%. Compliant ≥90%, Partial ≥60%, Non-Compliant <60%.

🖥️

CLI & Interactive

Full command-line interface with --demo, --interactive, and --verify modes. Scriptable for CI/CD pipelines.

🏢

Enterprise Ready

Persistent signing keys, batch processing, JSON metadata export, API integration ready. Free demo available.
